# Cucumber Test Automation Framework

A sample **Cucumber BDD (Behavior-Driven Development)** automation framework built using Java, Selenium, and Cucumber.

This project demonstrates how to write readable and maintainable automated test scenarios using Gherkin syntax and Cucumber step definitions.
